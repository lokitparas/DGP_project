MeshEdge const* nextAroundEndpoint ( int  i) const

Get the next edge when stepping counter-clockwise (when viewed from the "outside" of the mesh) around a specified endpoint, assuming the neighborhood is manifold.

This also assumes that face vertices/edges have consistent counter-clockwise winding order when viewed from the outside. On error, returns null.
